# Mining AI Optimization Project

Proyecto de simulación minera para optimización de decisiones operativas mediante IA.

## Contenido

- Generación de dataset minero
- Planificación de bloques
- Simulación de planta (lixiviación)
- Análisis exploratorio
- Base para modelos de Machine Learning y optimización

## Tecnologías
- Python
- Pandas
- Numpy
- Matplotlib

## Objetivo

Simular una operación minera subterránea de oro y plata para desarrollar modelos de decisión basados en datos.
